## 🛠 技术实现细节

### 使用的技术栈
- **Next.js 14** - App Router
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式系统
- **Lucide React** - 图标库
- **CSS动画** - 微交互效果

### 设计原则遵循

1. **大胆的美学选择**:
   - 每个功能都有独特的配色方案
   - 避免了通用的蓝/紫渐变
   - 使用了衬线字体、等宽字体等非标准选择

2. **精致的细节**:
   - 悬停效果
   - 过渡动画
   - 阴影和光效
   - 微交互

3. **独特的视觉元素**:
   - 赛博朋克的霓虹光效
   - 复古画廊的纸张纹理
   - 日记的水彩背景
   - 终端的代码高亮
